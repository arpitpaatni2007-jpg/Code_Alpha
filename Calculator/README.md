# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

- Basic operations: Addition, Subtraction, Multiplication, Division  
- Real-time input display  
- Clear (`C`) and Delete (`DEL`) functionality  
- Responsive design (works on mobile & desktop)  
- Simple and clean UI  

---

## 🛠️ Technologies Used

- HTML5 – Structure of the app  
- CSS3 – Styling and layout  
- JavaScript (Vanilla JS) – Logic and functionality  

---

## 📂 Project Structure

calculator-app/  
│── index.html  
│── style.css  
│── script.js  
│── README.md  

---

## ⚙️ How It Works

- User clicks buttons to input numbers and operators  
- JavaScript captures the input and updates the display  
- When `=` is pressed, the expression is evaluated  
- Result is shown instantly on the screen
